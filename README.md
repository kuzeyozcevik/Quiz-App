# Quiz-App
An interactive Python based quiz application built using Tkinter for the GUI and the Open Trivia Database API for real time questions.

This project demonstrates:
Object-Oriented Programming (OOP) principles,
API integration using requests,
GUI development with tkinter,
HTML entity decoding using html.unescape,
Code separation across multiple modules

True / False trivia questions fetched dynamically from OpenTDB,
Modern and responsive GUI,
Real-time score tracking,
Visual feedback for correct & incorrect answers,
Modular and readable project structure

Technologies Used:

Python 3,
tkinter,
requests,
Open Trivia DB API
